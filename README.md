# nodejs-cli-connect-four

Basic connect four game that can be played in your terminal.

```bash
Controls:
  Move cursor: <- & ->
  Insert piece: o

    x
[ ][ ][ ][ ][ ]
[ ][ ][ ][ ][ ]
[ ][ ][ ][ ][ ]
[ ][ ][y][y][y]
[x][x][y][x][x]

Player turn: x
```

## Usage

Clone repo:

```bash
git clone https://github.com/viglucci/nodejs-cli-connect-four.git
```

Enter directory:

```bash 
cd ejs-cli-connect-four
```

Run program:

```bash
node ./program.js
```
