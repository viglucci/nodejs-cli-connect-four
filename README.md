# nodejs-cli-connect-four

Basic connect four game that can be played in your terminal.

```bash
Controls:
  Move cursor: <- & ->
  Insert piece: o

       y
[ ][ ][ ][ ][ ][ ][ ]
[ ][ ][ ][ ][ ][ ][ ]
[ ][ ][ ][ ][ ][ ][ ]
[ ][ ][ ][x][ ][ ][ ]
[x][ ][ ][y][ ][ ][ ]
[x][y][x][y][ ][ ][ ]

Player turn: y
```

## Usage

Clone repo:

```bash
git clone https://github.com/viglucci/nodejs-cli-connect-four.git
```

Enter directory:

```bash 
cd nodejs-cli-connect-four
```

Run program:

```bash
node ./program.js
```
